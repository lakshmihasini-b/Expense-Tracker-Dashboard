# Expense-Tracker-Dashboard
#📌 Project Overview

The Expense Tracker Dashboard is a responsive web application developed using HTML, CSS, and JavaScript that helps users record and manage their daily financial transactions. It provides a simple interface for adding income and expenses, automatically calculates the current balance, and displays a transaction history. The project focuses on improving personal financial management while demonstrating core frontend development skills.

#🎯 Objective

The primary objective of this project is to:
- Help users monitor their daily income and expenses.
- Calculate the current balance automatically.
- Organize financial records in a simple and user-friendly interface.
- Practice DOM manipulation, event handling, and JavaScript logic.
#🚀 Key Features

💵 Income & Expense Management

- Add new income transactions.
- Add new expense transactions.
- Instant balance calculation.
  
#📊 Financial Summary

- Total Income
- Total Expenses
- Current Balance
  
#📝 Transaction History

- Displays all recorded transactions.
- Clearly distinguishes income and expenses using different colors.
- Updates automatically whenever a new transaction is added.

#📱 Responsive Design

- Works smoothly on desktop, tablet, and mobile devices.
- Clean and modern user interface.
